Number to word list converter based on T9 Dictionary

==================

BACKEND
------------------
```nvm use```
```npm install```
```npm start```

Server will run on localhost:5050

**TEST**
```npm test```

FRONTEND
------------------
```cd client```
```nvm use```
```npm install```
```npm run start```

Client will run on localhost:3000
